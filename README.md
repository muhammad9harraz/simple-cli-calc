# Calculator CLI

A simple command-line calculator built with Python.

## Features

- Add, subtract, multiply, divide
- Handles invalid input
- Division by zero protection

### Example Usage

-----Calculator-----
1. Add
2. Subtract
3. Multiply
4. Divide
5. Exit

Enter first number: 13
Enter second number: 7
Result:  20.0
